A plugin to simply embed Tape lead response forms in your WordPress posts and pages.

== Description ==

Tape enables you to send interactive video messages to customers' phones.

Delight customers with [Tape's video-over-SMS technology](https://www.trytape.com "Interactive video-over-SMS"). 98% open rates ensure your video-enhanced lead response, customer outreach, fundraising requests, and more have game-changing impact.

Using Tape can increase your leads conversion to sales by 50% in less than 10 minutes!. Here’s why:

* Highly customizable: Site owners can style the widget to match existing template and place it anywhere.
* FAST. Tape's WordPress widget is lightweight and built to scale, so it won’t slow down your site (which can hurt your Google rankings)
* Works on mobile! 

== Installation ==

1. Upload `tape-plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sign up at [TryTape.com](https://www.trytape.com "Automated Lead Response Forms") and create a campaign.
4. Use and customize our widget with your Campaign ID

***Usage***

`[tape campaignId="Your_Campaign_ID_Here" buttonColor="#aaa" buttonColorHover="#666" buttonText="Sign up"]`

***Parameters***

* **campaignId** : Tape Campaign ID : *required*
* **buttonColor** : Background color of submit button : *optional*
* **buttonColorHover** : Background color of submit button when hovered by mouse : *optional*
* **buttonText** : Text used on submit button : *optional*

== Frequently Asked Questions ==

= How do I get a Campaign ID? =

Sign up at [Tape's website](https://www.trytape.com "Customer outreach using video and sms") and create a campaign. All campaigns have IDs. Creating a new campaign will generate an ID for you to use.

= What happens if someone doesn't enter their name? =

Fortunately, names are optional. Your customers will receive a Tape over SMS whether they have shared with name or not.
